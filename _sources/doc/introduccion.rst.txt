Introducción
=============

Descripción General
--------------------

El webhook de SIREGOB-Odoo proporciona un sistema de notificaciones en tiempo real para la gestión de movimientos de
nómina entre los sistemas SIREGOB y Odoo.

Versión Actual
---------------
:Versión: 1.0
:Última actualización: 2024-09-25
:Estado: Desarrollo

Características Principales
----------------------------

* Notificaciones en tiempo real de movimientos de nómina
* Autenticación segura mediante JWT
* Verificación de integridad mediante HMAC
* Sistema de reintentos automáticos
* Monitoreo y logging integrado

Requisitos Previos
-------------------

* Python 3.8 o superior
* Acceso al sistema SIREGOB
* Credenciales de Odoo válidas
* Conexión a Internet estable
